# 🧮 Math Quiz Game (C++)

A console-based **Math Quiz Game** built using **C++**.  
The game allows players to test their math skills by choosing the number of questions, difficulty level, and operation type. At the end of the quiz, the player receives a detailed score summary.

---

## 📌 Features

- 📝 Choose the number of questions.
- 🎯 Select the difficulty level:
  - Easy
  - Medium
  - Hard
  - Mixed
- ➕ Choose the operation type:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Mixed
- 🎲 Randomly generated math questions.
- ✅ Instant feedback after each answer.
- 📊 Final score and quiz summary.
- 🔄 Play again without restarting the application.

---

## 🎮 How to Play

1. Enter the number of questions.
2. Choose the difficulty level.
3. Choose the math operation.
4. Answer all questions.
5. View your final score and performance summary.
6. Choose whether to play another game.
